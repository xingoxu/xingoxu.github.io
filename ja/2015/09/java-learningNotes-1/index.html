部署与安装

三步：

  1. 下载并安装JDK
  2. 配置环境变量
  3. 测试

注意点：

环境变量有三：

  1. JAVA_HOME 对应jdk根目录
  2. PATH 对应jdk\bin目录
  3. CLASSPATH 对应jdk\lib目录

使用IDE进行开发java程序

我选择eclipse

步骤：

  1. 创建java项目
  2. 创建程序包（一般为com.x）
  3. 创建源代码文件并编写
  4. 测试运行
